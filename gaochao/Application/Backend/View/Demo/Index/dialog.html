弹窗编辑数据

需要使用弹窗时，触发对象只需加上class=”dialog”即可

被弹出窗口中的form表单需要添加 class=”dialog_form”


示例：

第一步：调用相关js（__static__为大写），默认情况下该文件已被加载，无需再次加载

<script src="__static__/js/common.js"></script>

第二步：在需要的地方加上相关class

<!--
    a标签提供的弹窗属性有：
        title="弹窗标题，不填写弹窗将不会有标题出现";
        padding="为弹窗的内边距，值为数字：不需要添加px"
-->
<a  href="operate.html"  class="dialog">修改</a>

operate.html里的内容

<!--
    data-id是弹窗的事件触发对象
    当data-id的值为ok时，表示提交
    当data-id的值为cancel时，表示取消并关闭弹窗
-->
<form  action=""  method="post"  class="dialog_form">
    <input  type="hidden"  name="id"  value=""  />
    <input  type="submit"  class="btn submit ajax-post"  data-id="ok"  value="提交"  />
    <a  href=""  class="btn cancel"  data-id="cancel">取消并返回</a>
</form>
